How To
Purpose of the App
To demonstrate how to leverage the device’s biometric fingerprint sensors (Touch ID).
Prerequisites
•
SmartUX Studio installed and setup.
•
Physical device with a fingerprint sensor, and fingerprint identification setup on device.
Steps to run the Test app
1.
Install on a physical device
2.
Open the App
3.
Login with any username and password.
4.
Log out.
5.
Select the TouchID link.
6.
Re-authenticate using your fingerprint
7.
Use the Reset option to start over again.
